This Power BI dashboard provides an analysis of Airbnb listings across ten cities, focusing on growth trends, pricing patterns, ratings, user behavior, and host verification. Key insights include the concentration of listings in major cities such as Paris and New York, higher pricing for hotel rooms and entire homes, consistent ratings across locations, limited repeat reviews, and higher trust associated with verified hosts.
